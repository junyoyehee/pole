3. 타입 정의

Interface vs Type

객체 형태는 interface를 사용하고, 유니온/인터섹션 등은 type을 사용합니다.

// Good
interface User {
  id: string
  name: string
  role: UserRole
}

type Status = 'active' | 'inactive' | 'pending'

// Bad
type User = {
  id: string
  name: string
}

4. React 컴포넌트

함수 컴포넌트만 사용합니다. Props는 별도 interface로 정의합니다.

interface ButtonProps {
  label: string
  variant?: 'primary' | 'secondary'
  onClick: () => void
}

export function Button({ label, variant = 'primary', onClick }: ButtonProps) {
  return (
    <button className={styles[variant]} onClick={onClick}>
      {label}
    </button>
  )
}
