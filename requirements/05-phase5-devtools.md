# Phase 5: 개발 도구 & 대시보드

## 목표
개발팀의 일상 업무를 지원하는 도구와 팀 전체 현황을 파악할 수 있는 대시보드를 제공합니다.

---

## 5.1 팀 대시보드

### 설명
팀 전체 현황을 한눈에 파악하는 메인 대시보드

### 위젯 구성
```
┌─────────┬─────────┬─────────┬─────────┐
│ 진행중   │ 리뷰대기 │ 이번주   │ 팀원    │
│ 프로젝트 │  업무    │ 마감업무 │ 현황    │
│   3개   │  7건    │   5건   │ 18/20  │
├─────────┴─────────┴─────────┴─────────┤
│                                       │
│   프로젝트별 진행률 차트                  │
│                                       │
├───────────────────┬───────────────────┤
│ 최근 디자인 리뷰    │ 최근 공지사항      │
│ - 리뷰 1          │ - 공지 1          │
│ - 리뷰 2          │ - 공지 2          │
│ - 리뷰 3          │ - 공지 3          │
├───────────────────┼───────────────────┤
│ 최근 업무 활동      │ 팀원 근무 상태     │
│ - 활동 1          │ 🟢 김민수 근무중   │
│ - 활동 2          │ 🟡 이영희 회의중   │
│ - 활동 3          │ 🔴 박철수 휴가    │
└───────────────────┴───────────────────┘
```

### 통계 카드
| 카드 | 데이터 |
|------|-------|
| 진행중 프로젝트 | 현재 진행중인 프로젝트 수 |
| 리뷰 대기 | 디자인 리뷰 + 업무 리뷰 대기 건수 |
| 이번 주 마감 | 이번 주 마감인 업무 수 |
| 팀원 현황 | 현재 근무중 / 전체 팀원 |

### 위젯
- 프로젝트별 진행률 (프로그레스 바)
- 최근 디자인 리뷰 (3건)
- 최근 공지사항 (3건)
- 최근 업무 활동 피드
- 팀원 근무 상태 목록

---

## 5.2 코드 스니펫 공유

### 설명
팀 내에서 유용한 코드 스니펫을 공유하고 검색하는 도구

### 스니펫 등록
| 항목 | 설명 |
|------|------|
| 제목 | 스니펫 이름 |
| 설명 | 용도, 사용법 설명 |
| 언어 | TypeScript, CSS, SQL, Shell 등 |
| 코드 | 코드 내용 (구문 강조) |
| 태그 | 검색용 태그 (예: react, hook, api) |
| 관련 Figma | Figma 컴포넌트 연결 (선택) |

### 기능
- 코드 구문 강조 (Syntax Highlighting)
- 원클릭 복사
- 태그/언어별 필터
- 검색
- 좋아요 (유용한 스니펫 표시)
- Figma 연결 시 디자인 미리보기 함께 표시

## 5.3 배포 현황

### 설명
프로젝트의 배포 상태를 추적하는 간단한 보드

### 배포 항목
| 항목 | 설명 |
|------|------|
| 프로젝트 | 배포 대상 프로젝트 |
| 환경 | Development / Staging / Production |
| 버전 | 배포 버전 (예: v1.2.3) |
| 배포자 | 배포 담당자 |
| 배포일시 | 배포 시간 |
| 상태 | 성공 / 실패 / 롤백 |
| 변경사항 | 주요 변경 내용 (마크다운) |

### 배포 보드 뷰
```
┌─────────────────────────────────────┐
│ 프로젝트 A                          │
│ DEV: v1.3.0 ✅  STG: v1.2.5 ✅  PROD: v1.2.3 ✅ │
├─────────────────────────────────────┤
│ 프로젝트 B                          │
│ DEV: v2.1.0 ✅  STG: v2.0.8 ⚠️  PROD: v2.0.5 ✅ │
└─────────────────────────────────────┘
```
- 프로젝트별 환경 상태 한눈에 표시
- 배포 히스토리 (시간순)
- 실패 시 알림

## 5.4 알림 시스템

### 설명
인트라넷 전반의 알림을 통합 관리

### 알림 대상 이벤트
| 이벤트 | 수신자 |
|--------|--------|
| 디자인 리뷰 요청 | 지정된 리뷰어 |
| 디자인 리뷰 피드백 | 리뷰 요청자 |
| 업무 할당 | 담당자 |
| 업무 상태 변경 | 관련 팀원 |
| 핸드오프 등록 | 담당 개발자 |
| 새 공지사항 | 전체 팀원 |
| 회의록 액션 아이템 | 담당자 |
| 댓글 알림 | 글 작성자 |
| 업무 마감 임박 (D-1) | 담당자 |

### 알림 UI
- 헤더 벨 아이콘 + 뱃지 (읽지 않은 수)
- 알림 드롭다운 패널
- 전체 알림 목록 페이지
- 알림 읽음 처리
- 알림 설정 (이벤트별 ON/OFF)
