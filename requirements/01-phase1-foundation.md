# Phase 1: 기반 구축

## 목표
인트라넷의 공통 레이아웃, 인증, 팀원 관리 등 전체 서비스의 기반을 구성합니다.

---

## 1.1 레이아웃

### AppShell 구조
```
┌──────────────────────────────────────────────┐
│ [≡] Pole       🔍 검색     🔔  🌙  [프로필] │ ← 헤더 (60px)
├────────┬─────────────────────────────────────┤
│ 사이드바 │                                    │
│        │         메인 콘텐츠                  │
│ 메뉴    │                                    │
│        │                                    │
│ (240px)│                                    │
├────────┴─────────────────────────────────────┤
```

### 사이드바 메뉴 구조
```
── 메인
   ├── 대시보드
   └── 내 업무
── Figma
   ├── 디자인 허브
   ├── 디자인 리뷰
   └── Figma 탐색기
── 프로젝트
   ├── 프로젝트 목록
   └── 타임라인
── 소통
   ├── 공지사항
   ├── 팀 게시판
   ├── 회의록
   └── 위키
── 도구
   ├── 코드 스니펫
   └── 배포 현황
── 관리 (관리자만)
   ├── 팀원 관리
   └── 시스템 설정
```

### 반응형
- 데스크탑 (1280px+): 사이드바 펼침
- 태블릿 (768px~1279px): 사이드바 접힘 (아이콘만)
- 모바일 (~767px): 사이드바 오버레이

## 1.2 인증

### 로그인
- 이메일/비밀번호 로그인
- 로그인 유지 (Remember me)
- 비밀번호 찾기 (이메일 발송)

### 계정 관리
- 초기 계정은 관리자가 생성 (초대 방식)
- 자체 가입 없음 — 팀원 초대 링크로만 가입
- 역할: MEMBER, LEAD, ADMIN

### 권한 체계
| 역할 | 설명 | 접근 범위 |
|------|------|----------|
| MEMBER | 일반 팀원 | 기본 기능 전체 |
| LEAD | 팀 리드 | + 프로젝트 관리, 업무 할당 |
| ADMIN | 관리자 | + 팀원 관리, 시스템 설정 |

## 1.4 팀원 목록 & 조직도

- 팀 리드 표시

## 1.5 테마 & 공통

### 테마
- 다크/라이트 모드 전환
- 한국어 폰트 (Pretendard)
- Mantine 테마 커스터마이징

### 공통 컴포넌트
| 컴포넌트 | 설명 |
|---------|------|
| PageHeader | 페이지 제목 + 브레드크럼 + 액션 |
| DataTable | 정렬, 페이지네이션, 검색 테이블 |
| EmptyState | 빈 상태 안내 |
| ConfirmModal | 확인/삭제 모달 |
| StatusBadge | 상태 뱃지 (파트, 역할, 진행상태 등) |
| UserCard | 팀원 카드 (아바타 + 이름 + 직책) |

### 에러/로딩
- 404 Not Found 페이지
- Error Boundary
- 로딩 스켈레톤
- 토스트 알림 (Mantine Notifications)
